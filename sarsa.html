Gridworld

Python

Gridworld is an implementation of the SARSA(state-action-reward-state-action) machine learning algorithm built with Python using Tkinter to build for the UI. Gridworld uses a 20 x 20 grid with a single reward state. An agent is started at a random state then must move up, down, left, or right until it finds the reward state. After finding the reward the agent is placed in a new random location in the grid and must use previous findings to help it navigate to the reward state again.
